# 🔐 Password Strength Checker

A simple yet powerful web application built with **Streamlit** to evaluate the strength of passwords and help users generate strong, secure ones.

## 🚀 Features

- ✅ Real-time password strength evaluation
- ⚠️ Checks against a list of common weak passwords
- 🛡️ Provides actionable feedback to improve password strength
- 🔒 Strong password generator with all security requirements
- 🎈 Interactive and user-friendly interface with Streamlit

## 📋 Password Evaluation Criteria

A password is scored based on:
- Minimum length (8 characters)
- At least one **uppercase** letter (A-Z)
- At least one **lowercase** letter (a-z)
- At least one **digit** (0-9)
- At least one **special character** (!@#$%^&*)

## 🔧 Technologies Used

- Python 🐍
- Streamlit 🌐
- Standard Libraries: `random`, `string`
